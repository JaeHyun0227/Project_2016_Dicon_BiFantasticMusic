===============================
확인창 UI
===============================

원리를 잠깐 설명하자면

말풍선 왼쪽에는 캐릭터를 세울것이며
말풍선 안쪽에는 캐릭터에 맞는 대사를 끼울 계획임 !

Y를 누르면 전투가 실행되고 N을 누르면 빠져나오는 방식 

Y랑 N 버튼은 각각 그림에 써잇는 것 처럼 초록색과 빨간색으로 처리해주면 되고,
두 버튼 둘다 누른거 각각 줘야되므로 

Y 버튼과 N 버튼은 따로따로 줘야됨


결론 :: 5개 파일 줘야됨

(1) 말풍선 + 두버튼 제외 UI
(2) Y 버튼 눌린거
(3) Y 버튼 안눌린거
(4) N 버튼 눌린거
(5) N 버튼 안눌린거
